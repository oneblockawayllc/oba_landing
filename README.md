# One Block Away Website

A Next.js website showcasing AI Implementation & Orchestration services for downtown Apex and RTP region businesses.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

To create a production build:

```bash
npm run build
```

To start the production server:

```bash
npm start
```

## Project Structure

- `app/` - Next.js app directory with layout and pages
- `components/` - React components for different sections
- `public/` - Static assets

## Features

- Responsive design matching Wix site structure
- Smooth scrolling navigation
- Contact form with validation
- Focus on AI Implementation & Orchestration
- Geographic focus on Apex/RTP region
