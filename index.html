Void & Signal

Design System Documentation

A precision design system built on deep space depth, electric signals, and AI-native aesthetics. Dark foundations with cyan-violet energy.

00 Foundations

System Principles

Design Philosophy

  • Deep void foundations — never pure black, always with depth
  • Signal colors (cyan) for primary actions and focus states
  • Pulse colors (violet) for accents and secondary emphasis
  • Glow effects to create electric presence
  • Precision over decoration — every element has purpose

Technical Standards

  • 4.5:1 contrast minimum for all text
  • Focus indicators use signal glow
  • 44px minimum touch targets
  • Grid-based spacing (4px base unit)
  • Reduced motion support for all animations
